## Setup
Clone this repo to your desktop and run `npm install` to install all the dependencies.

## Install

```sh
npm install
```

## Usage

Add a .env file and update the values/settings to your own

```sh
npm start
```

## Run tests

```sh
npm test
```

## Debugging

```sh
npm run debug
```
